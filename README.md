# harry potter api project

## descripcion
aplicacion web que consume la api publica de harry potter y muestra una lista de personajes con imagen, nombre, casa y actor. permite filtrar por casa y ver un detalle individual.

## funcionalidades
- consumo de api con fetch
- renderizado dinamico con javascript
- filtro por casa
- vista de detalle usando parametros en la url

## estructura
- index.html: lista principal de personajes
- detalles.html: vista individual del personaje
- js integrado en cada archivo

## funcionamiento
1. se hace una peticion a la api.
2. se filtran personajes con imagen.
3. se limitan a 25 resultados.
4. se renderizan en el dom.
5. al hacer click se redirige a detalles.html con el nombre en la url.

## tecnologias

- css
- javascript
- api publica hp-api
